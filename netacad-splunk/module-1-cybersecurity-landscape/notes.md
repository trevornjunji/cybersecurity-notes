## Key Terms to Remember
| Term | Definition |
|------|------------|
| Confidentiality | Ensuring data is accessible only to authorised users |
| Integrity | Ensuring data is accurate and untampered |
| Availability | Ensuring systems and data are accessible when needed |
| Threat | Any potential danger to an information system |
| Vulnerability | A weakness that can be exploited by a threat |
| Risk | The likelihood and impact of a threat exploiting a vulnerability |
| SOC | Security Operations Centre — team monitoring for threats |
| Threat Actor | Individual or group responsible for a cyber incident |

## How This Connects to Real Work
- SOC analysts use the CIA Triad daily to classify 
  the impact of security incidents
- Identifying threat actors helps analysts prioritise 
  response efforts based on sophistication and intent
