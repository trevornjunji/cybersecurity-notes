## Key Concepts
- MITRE ATT&CK framework — tactics vs techniques
- Cyber Kill Chain — seven stages of an attack
- Types of malware and their behaviour
- Social engineering attack methods
- Common attack vectors

## Important Frameworks & Models
- MITRE ATT&CK — maps real-world adversary tactics 
  and techniques used across the attack lifecycle
- Cyber Kill Chain — Reconnaissance, Weaponisation, 
  Delivery, Exploitation, Installation, 
  Command & Control, Actions on Objectives

## Key Terms to Remember
| Term | Definition |
|------|------------|
| MITRE ATT&CK | Framework mapping adversary tactics and techniques |
| Tactic | The adversary's goal at a specific stage of attack |
| Technique | How the adversary achieves that goal |
| Kill Chain | Model describing the stages of a cyberattack |
| Malware | Malicious software designed to damage or gain access |
| Virus | Malware that attaches to files and self-replicates |
| Worm | Self-replicating malware that spreads without user action |
| Trojan | Malware disguised as legitimate software |
| Ransomware | Malware that encrypts files and demands payment |
| Phishing | Fraudulent emails designed to steal credentials |
| Spear Phishing | Targeted phishing aimed at a specific individual |
| Vishing | Voice-based phishing via phone calls |
| Attack Vector | Path or method used by an attacker to access a system |

## How This Connects to Real Work
- SOC analysts map incoming alerts to MITRE ATT&CK 
  techniques to understand attacker behaviour
- Kill Chain analysis helps analysts determine how 
  far into an attack an adversary has progressed
- Recognising social engineering protects organisations 
  from their most common attack entry point
